#### 设置字符编码集
```console
:set encoding=utf-8
:set fileencodings=ucs-bom,utf-8,cp936
:set fileencoding=gb2312
:set termencoding=utf-8
```

#### 显示(取消)行号
```console
:set number     显示行号
:set nonumber   取消行号
```

#### tab和space转换
```console
set shiftwidth=4    当按回车下一行时，下一行留4个长度的space
set ts=4  tab使用4个space长度
```
