##### 查看elf文件头
```console
readelf -h xxx
```

##### 查看elf文件各个section
```console
readelf -S xxx (按照段表索引号排列各个section)
objdump -h xxx (按照相对elf文件的起始位置排列各个section)
```

##### 查看elf文件的符号表
```console
nm xxx
readelf -s xxx (输出更为清晰)
```

##### 根据elf文件符号反查C++命名
```console
c++filt _ZN1N1C4funcEi (输出为N::C::func(int)) 
```

##### 查看ld默认ld script
```console
ld --verbose
```

##### 查看反汇编内容
```console
objdump -d xxx
```

#### 查看重定位表内容
```console
objdump -r xxx
```
